## Описание приложения

Описание приложения (универсальное)

 Генератор паролей — быстро и безопасно

Это простое и удобное приложение для создания надёжных паролей.
Пользователь может выбрать длину пароля, использовать буквы, цифры и специальные символы. Сгенерированный пароль отображается на отдельном экране и может быть скопирован в буфер обмена одним нажатием.

Приложение не требует регистрации, не сохраняет данные и работает полностью локально, что обеспечивает безопасность и конфиденциальность.
